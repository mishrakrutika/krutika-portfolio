## Chosen Approach: Digital Blueprint

### Design Movement
Neo-Brutalism meets Technical Minimalism — clean geometric shapes, bold typography, and purposeful dark backgrounds with electric accent colors.

### Core Principles
1. **Technical Precision**: Every element feels deliberate and engineered, like well-written code
2. **Dark Elegance**: Deep backgrounds create focus and reduce eye strain for recruiters browsing late
3. **Accent Energy**: Vibrant teal-cyan highlights draw attention to key sections and CTAs
4. **Generous Breathing Room**: Ample whitespace communicates confidence and clarity

### Color Philosophy
- **Primary Background**: Deep charcoal (#0F1419) — not pure black, but rich enough to feel premium
- **Secondary Background**: Slightly lighter slate (#1A2332) — for cards and section differentiation
- **Accent**: Electric teal (#06D6A0) — represents growth, energy, and modern tech
- **Text Primary**: Warm white (#F0F4F8) — easy on the eyes against dark backgrounds
- **Text Secondary**: Muted gray-blue (#8899AA) — for descriptions and secondary content
- **Emotional intent**: Conveys professionalism, technical depth, and forward momentum

### Layout Paradigm
Asymmetric single-page scroll with floating sidebar navigation. Sections use alternating left/right emphasis to create visual rhythm. Cards use slight 3D perspective on hover. No centered hero — text sits left with decorative elements on the right.

### Signature Elements
1. **Glitch-border cards**: Subtle animated border glow on project cards using CSS gradient borders
2. **Terminal-style section labels**: Each section heading prefixed with `/>` in monospace accent color
3. **Floating code snippets**: Decorative code blocks as background elements in the hero

### Interaction Philosophy
- Hover states use color shifts and subtle scale transforms (scale: 1.02)
- Navigation dots pulse when scrolling to a section
- Skill badges have a "typing" entrance animation
- Contact form has a subtle glow on focus

### Animation
- Scroll-triggered fade-up with 0.3s delay between elements
- Hero text staggers letter by letter (30ms per letter)
- Project cards lift on hover with shadow intensification
- Background gradient slowly shifts hue over 20s cycle
- Skill bars fill from 0 to width with ease-out easing
- All animations respect `prefers-reduced-motion`

### Typography System
- **Headings**: Space Grotesk (700 for h1, 600 for h2, 500 for h3) — geometric, modern, technical
- **Body**: DM Sans (400 regular, 500 medium) — clean, highly readable
- **Monospace**: JetBrains Mono (400) — for code snippets, section labels, and technical details
- **Scale**: h1: 3.5rem, h2: 2.5rem, h3: 1.5rem, body: 1rem, small: 0.875rem

### Brand Essence
Krutika Mishra — a focused, driven frontend developer building real-world applications and hungry to grow through hands-on internship experience.
**Personality**: Focused, Eager, Technical

### Brand Voice
- Headlines: Direct, confident, action-oriented
- CTAs: Clear and specific, not vague
- Microcopy: Friendly but professional
- **Examples**:
  - "Building interfaces that work. Seeking opportunities that matter."
  - "Let's build something together — reach out."

### Wordmark & Logo
A minimalist monogram "KM" inside a rounded square, with the K's vertical stroke extended downward to form a cursor line — blending identity with the developer aesthetic. Bold, no-serif, white on dark background.

### Signature Brand Color
**Teal Glow #06D6A0** — the unmistakable accent that appears in borders, buttons, icons, and highlights throughout the site.
